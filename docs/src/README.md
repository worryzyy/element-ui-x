## 快速上手

```bash
# 安装
npm install vue-element-ui-x

# 引入
import Vue from 'vue'
import ElementUI from 'element-ui'
import 'element-ui/lib/theme-chalk/index.css'
import ElementUIX from 'vue-element-ui-x'

Vue.use(ElementUI)
Vue.use(ElementUIX)
```

查看 [快速上手](/guide/quickstart) 了解更多信息。
