# Thinking 思考动画组件

## 功能说明

思考动画组件，用于展示 AI 思考过程的动画效果，支持以下特性：

- 可自定义动画样式和颜色
- 支持多种思考动画效果
- 可配置动画持续时间
- 支持动画中断和恢复
