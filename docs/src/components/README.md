# Element-X 示例

- [Typewriter 打字机](./typewriter/) - 打字机效果组件的完整示例和用法

## 预览示例

你可以通过以下方式启动示例预览：

```bash
# 启动文档开发服务器，查看组件文档和示例
npm run dev:docs

# 启动demo示例服务器，查看完整应用示例
npm run dev:demo
```
