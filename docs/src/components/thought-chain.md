# ThoughtChain 思维链组件

## 功能说明

思维链组件，用于展示 AI 思考过程的步骤和逻辑链，支持以下特性：

- 可折叠的思考步骤
- 支持 Markdown 内容渲染
- 步骤之间的逻辑关联展示
- 可交互的思考过程展示
