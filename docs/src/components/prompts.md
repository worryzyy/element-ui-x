# Prompts 提示词库组件

## 功能说明

提示词库组件，用于管理和使用 AI 提示词模板，支持以下特性：

- 提示词分类和搜索
- 提示词模板编辑和保存
- 变量插入和替换功能
- 一键应用提示词到对话
