# Sender 发送框组件

## 功能说明

发送框组件，用于用户输入和发送消息，支持以下特性：

- 自动调整高度的输入区域
- 支持快捷键发送（Enter/Ctrl+Enter）
- 支持表情和附件上传
- 可自定义操作按钮
