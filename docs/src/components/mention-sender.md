# MentionSender 提及输入框组件

## 功能说明

提及输入框组件，用于支持在输入过程中@提及用户或插入特殊标记，支持以下特性：

- @提及用户功能
- 自动补全和建议
- 支持自定义提及项渲染
- 与 Sender 组件集成
