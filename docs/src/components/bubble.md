# Bubble 聊天气泡组件

## 功能说明

聊天气泡组件，用于展示对话内容，支持以下特性：

- 自定义气泡样式和颜色
- 支持用户和机器人两种角色
- 自适应内容宽度
- 支持 Markdown 内容渲染
