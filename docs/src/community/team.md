## 加入我们

我们欢迎各种形式的贡献，如果你希望加入 Element-UI-X 团队，可以通过以下方式：

1. 在 GitHub 上提交有价值的 Pull Request
2. 参与社区讨论，提供有价值的反馈
3. 帮助改进文档和编写教程
4. 联系核心团队成员表达加入意愿
