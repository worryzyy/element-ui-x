# Element UI X 文档

欢迎使用 Element UI X 组件库文档

## 快速开始

```bash
npm install vue-element-ui-x
```

```javascript
import Vue from 'vue';
import ElementUI from 'element-ui';
import ElementUIX from 'vue-element-ui-x';

Vue.use(ElementUI);
Vue.use(ElementUIX);
```

## 特性

- 基于 Element UI 2.x
- 企业级 AI 聊天组件
- 主题可定制
