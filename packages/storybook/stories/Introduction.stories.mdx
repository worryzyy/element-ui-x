# Element UI X

基于Vue 2 + Element UI的企业级AI聊天组件库

## 简介

Element UI X 是一个专为AI聊天应用而设计的Vue组件库，基于Element UI构建，提供了丰富的聊天界面组件和交互功能。

## 组件概览

### 核心组件

- **Bubble** - 聊天气泡组件
- **BubbleList** - 聊天气泡列表
- **Conversations** - 对话列表
- **Sender** - 消息发送器
- **Welcome** - 欢迎组件

### 扩展组件

- **Attachments** - 附件组件
- **FilesCard** - 文件卡片
- **Prompts** - 提示组件
- **Thinking** - 思考动画
- **ThoughtChain** - 思维链
- **Typewriter** - 打字机效果

## 特性

- 🎨 基于Element UI设计体系
- 🌍 国际化支持
- 📱 响应式设计
- ⚡ Vue 2兼容
- 🎯 专为AI聊天场景优化
- 🔧 高度可定制

## 开始使用

在左侧导航栏中选择任意组件开始探索。每个组件都包含详细的文档、交互示例和代码片段。
