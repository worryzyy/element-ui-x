# Element UI X

基于 Vue 2 + Element UI 的企业级 AI 聊天组件库

## 简介

Element UI X 是一个专为 AI 聊天应用而设计的 Vue 组件库，基于 Element UI 构建，提供了丰富的聊天界面组件和交互功能。

## 组件概览

- **Bubble** - 聊天气泡组件
- **BubbleList** - 聊天气泡列表
- **Conversations** - 对话列表
- **Sender** - 消息发送器
- **Welcome** - 欢迎组件
- **Attachments** - 附件组件
- **FilesCard** - 文件卡片
- **Prompts** - 提示组件
- **Thinking** - 思考动画
- **ThoughtChain** - 思维链
- **Typewriter** - 打字器
